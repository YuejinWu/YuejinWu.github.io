AI Platform部分技术栈

Yuejin Wu2025-12-082025-12-09

以下是为我零基础新手量身定制的容器平台技术栈学习路线。

🎯 零基础学习路线图（总览）

预计周期：3-4个月（按每周10小时学习时间）

📘 第一阶段：基础认知与Linux (1-2周)

目标：建立计算机基础认知，学会基本的Linux操作

学习内容：

  1. 计算机基础概念

    • 什么是操作系统、进程、内存、硬盘
    • 什么是IP地址、端口、网络协议
    • [B站搜索]：“计算机基础入门” 系列视频

  2. Linux基础命令（最重要！）

    • 如何在命令行中移动、查看文件
    • 如何安装软件、管理权限
    • 推荐学习资源：
      • [B站]：搜索“Linux基础命令10小时”
      • [网站]：菜鸟教程Linux教程
      • 动手练习：在Windows上安装WSL2，或Mac使用终端

每日任务示例：

  • Day1：学习 ls, cd, pwd 命令
  • Day2：学习 mkdir, rm, cp 命令
  • Day3：学习 cat, vim, nano 编辑文件
  • Day4：学习 ps, top 查看进程
  • Day5：学习 chmod, chown 权限管理

🐳 第二阶段：Docker核心技能 (3-4周)

目标：掌握Docker基本使用，理解容器概念

学习路径：

  1. Docker是什么？（第1周）

    • 看这个视频：[B站]“Docker通俗易懂解释”
    • 安装Docker Desktop（Windows/Mac一键安装）
    • 运行第一个容器：docker run hello-world

  2. Docker基础操作（第2周）

    • 学习核心命令：
    • 完成一个小练习：用Docker运行一个Nginx网站

  3. Dockerfile与镜像制作（第3周）

    • 理解镜像和容器的关系
    • 学习编写简单的Dockerfile
    • 自己制作一个包含Python环境的镜像

  4. 数据持久化（第4周）

    • 理解Volume的概念
    • 学习将本地文件夹挂载到容器内
    • 练习：创建一个MySQL容器，将数据保存在本地

推荐资源：

  • [视频]：B站“Docker入门到实战”
  • [交互式学习]：Docker官方教程（有中文）
  • [书籍]：《Docker入门与实践》（电子版）

💾 第三阶段：存储与网络基础 (2-3周)

目标：理解平台中的存储和网络设计

学习内容：

  1. NFS基础（第1周）

    • 什么是NFS（网络文件系统）
    • 为什么需要共享存储
    • 简单搭建一个NFS服务体验

  2. 理解平台存储架构

    • 重新阅读文档中的“容器磁盘 vs NFS云盘”
    • 画图理解：容器内部 /root 目录是如何挂载NFS的
    • 理解为什么conda环境可以共享

  3. 网络基础（第2-3周）

    • 什么是IP、端口、网关
    • Docker网络基础：桥接网络、端口映射
    • 理解 -p 8080:80 是什么意思
    • 学习简单的网络故障排查

动手实验：

  • 在本地搭建两个虚拟机，配置NFS共享
  • 创建一个Docker容器，挂载NFS共享目录

🏗️ 第四阶段：平台实战与理解 (2-3周)

目标：将所学应用到实际平台理解中

学习步骤：

  1. 平台架构图绘制

    • 根据文档，画出平台的简化架构图
    • 标记出：用户 → Web界面 → Gateway → Host → Container → NFS
    • 理解数据流向

  2. 实际操作平台

    • 申请一个测试容器
    • 在容器内安装软件（用apt）
    • 在/root目录下创建文件，理解NFS共享
    • 用conda创建一个Python环境

  3. 资源监控初识

    • 学习使用 htop, nvidia-smi, df -h 等命令
    • 理解平台文档中的“CPU、内存、GPU监控”

  4. 问题排查基础

    • 容器启动失败怎么办？
    • 网络不通怎么办？
    • 磁盘空间不足怎么办？

🚀 第五阶段：扩展与进阶 (持续学习)

目标：深入理解平台高级特性

可选学习方向：

  1. 容器编排：学习Docker Compose、Kubernetes基础
  2. 监控系统：学习Prometheus + Grafana
  3. CI/CD：学习GitLab CI或GitHub Actions
  4. 安全基础：学习容器安全、权限控制

📚 学习资源推荐（零基础友好）

视频教程（B站）：

  1. 【Linux】“Linux基础命令全集” - 黑马程序员
  2. 【Docker】“Docker容器技术全集” - 尚硅谷
  3. 【网络】“计算机网络基础” - 湖科大教书匠

网站/文档：

  1. 菜鸟教程（runoob.com）- Linux/Docker基础
  2. Docker官方文档（有中文）- 最权威
  3. MDN Web文档 - 学习网络基础知识

实践平台：

  1. 本地：Docker Desktop + WSL2
  2. 在线实验：Katacoda（免费Docker实验环境）
  3. 云服务器：阿里云/腾讯云学生优惠（最低配置即可）

🎯 具体学习计划表（前2个月）

周数 学习重点 每日任务（约1-2小时） 完成标志
第1周 Linux基础 学习5个命令+练习 能在命令行中自如移动、查看文件
第2周 Docker安装与运行 安装Docker，运行3种不同容器 成功运行Nginx、Redis、MySQL容器
第3周 Docker命令精通 掌握10个核心命令 能独立管理容器生命周期
第4周 Dockerfile实践 编写2个Dockerfile 成功构建自定义镜像
第5周 数据持久化 练习Volume挂载 实现容器数据持久化保存
第6周 NFS基础 搭建简单NFS服务 实现两台机器文件共享
第7周 平台理解 画出平台架构图 能向别人解释平台工作原理
第8周 平台实战 在平台上完成一个任务 成功创建容器、安装软件、保存数据

❓ 遇到困难怎么办？

  1. 谷歌/百度是你的朋友：搜索“docker如何xxx”
  2. 加入技术社区：Stack Overflow、SegmentFault、知乎
  3. 问问题技巧：描述你做了什么、期望结果是什么、实际结果是什么、错误信息是什么
  4. 先尝试再问：自己尝试解决30分钟，解决不了再问

💡 最重要建议

  1. 动手 > 看视频 > 看书：技术学习必须动手
  2. 不要怕犯错：容器可以随意删除重建
  3. 从简单开始：先跑起来，再理解原理
  4. 定期复习：每周回顾一次学过的内容
  5. 找同伴：找个一起学习的伙伴互相督促

现在就开始：今天就在你的电脑上安装Docker，运行 docker run hello-world！
